Odoo Arabic (Right to Left)


Go to Settings -> Load Arabic Translation.


Go to Settings -> Activate developer mode.


Go to Settings ->Languages ->Go to Arabic language. Set direction - Right-to-Left.


Click on current user(any other user from users) -> Preferences. Change language to Arabic.








After reload/ refresh, the window is changed to Right to Left format.
