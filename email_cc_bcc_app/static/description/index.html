Customer Email CC BCC in Odoo

Odoo Email CC and BCC is specially designed to send email to multiple partners in CC and multiple partners in BCC of mail. Users can select partners for CC, BCC and also make BCC or CC visible in emails by enabling them from general settings, according to the requirement.

Select default CC and BCC in general settings.

User can select multiple partners for default CC and default BCC in general settings.

Send mail to the Customers with CC.

User can send mail to the customers with selected CC in general settings.

Send mail to the Customers with BCC.

User can send mail to the customers with selected BCC in general settings.
